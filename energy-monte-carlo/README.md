\# Energy Price Monte Carlo Simulation  
  
This project models energy prices using historical data and Monte Carlo simulation.  
  
\## Overview  
- Downloads real data from Yahoo Finance  
- Computes log returns and volatility  
- Simulates future price paths using Geometric Brownian Motion  
- Estimates expected price and confidence intervals  
  
\## Technologies  
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- yfinance
